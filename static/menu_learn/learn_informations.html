Règles de fonctionnement
Le connecteur actif sera utilisé depuis le catalogue formation pour publier ou mettre à jour une fiche dans le LMS. L’export HTML manuel reste disponible dans tous les cas.
L’ApiID est stocké côté serveur sous forme chiffrée. Il n’est jamais renvoyé dans l’interface après enregistrement.
